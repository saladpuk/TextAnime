# Text Anime

## How to use
Open commandline and type `dotnet run` with `arguments` see the description below.

```csharp
dotnet run [text] [Commands]
```
> **Text**  
The text that you want to used to play the animation.

> **Commands**
> * fs - Wording will fall straight.
> * bl - Wording will bounce to the left.
> * br - Wording will bounce to the right.
> * be - Wording will bounce to the edge.
> * sl - Wording will slide to the left.
> * sr - Wording will slide to the right. 

## Example
```csharp
dotnet run saladpuk fs bl br sl sr
```
